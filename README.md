## 배경

- 의사결정자(보직자/행정)는 AI 비전문가
- "AI로 가시적 성과를 보여라" → 단기적·홍보용 산출물 요구
- 구체적 아이디어 없이 "어떻게든 뭐라도" 상태
- AI 교육 중심 제안은 거절됨 (가시적 성과가 아니라서)
- 대상 청중도 외부 AI 비전문가 ("우리 대학은 AI를 활용한다" 인상)
