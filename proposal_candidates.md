# AI University Project — 초기 제안 후보군

## 후보 1: "학과별 AI 학습 어시스턴트" (다국어 RAG 챗봇) ✅ 채택

**개요**: 각 학과의 교육과정·입학안내·FAQ를 학습한 AI 어시스턴트. 중국어/영어/한국어로 질의응답.

| 평가 | 점수 | 근거 |
|---|---|---|
| A. 비전문가 임팩트 | ★★★★★ | "AI가 우리 학과에 대해 답한다" — 체험형 |
| B. AI 활용도 | ★★★★☆ | RAG + 다국어 LLM = 진짜 AI 기술 |
| C. 구현 가능성 | ★★★★☆ | Gemini API + 학과 PDF → 2-3주 |
| D. 확장성 | ★★★★★ | 학과별 데이터만 바꾸면 8개 복제 |
| E. Google 활용 | ★★★★★ | Gemini API 직접 사용 |

**채택 이유**: 즉시 체험 가능 + ChatGPT 차별화 + 홍보 후 실사용 전환 가능

| C. 구현 가능성 | ★★★★★ | Gemini API + 웹폼, 1-2주 |
| D. 확장성 | ★★★★☆ | 학과별 템플릿 |
| E. Google 활용 | ★★★★☆ | Gemini API |
