AI 학습 어시스턴트

다국어 RAG 기반 학과 상담 챗봇 (환각 방어형)

유학생이 모국어로 질문하면 학과 데이터를 기반으로 답변합니다. "모르는 것은 지어내지 않고 행정실로 연결"하는 3단계 방어 로직(민감 키워드 차단, 유사도 검증, 불확실성 필터)을 탑재하여 법적/행정적 리스크를 대폭 감소시킵니다.

🛡️ 환각(Hallucination) 방어 ⚠️ 민감 질문 자동 차단 🌐 3개 국어 지원 📞 부서 연락처 자동 안내
기술 Gemini API (검증용) → 로컬 LLM
데이터 검수된 Markdown 문서 한정
운영 비용 Phase 2 전환 시 토큰 비용 없음
안전성 면책 조항 및 로그 수집

학과 소개 AI 포탈

데이터 기반 자동 빌드 정적 웹페이지

학과 데이터를 AI가 다국어 소개글로 재구성합니다. 실시간 API 호출로 인한 비용과 지연을 없애기 위해, '데이터 변경 시에만 정적(Static)으로 사전 빌드'하여 자체 서버나 무료 호스팅에 배포하는 유지보수 최소화 아키텍처를 적용합니다.

⚡ 사전 빌드 (SSG) 💰 서버 유지비 최소화 📖 다국어 자동 생성 🔒 정적 파일 배포
기술 정적 웹 (HTML/CSS/JS)
업데이트 데이터 수정 시 자동 반영
운영 비용 빌드 시에만 API 비용 발생
연동 챗봇(어시스턴트) 연결 링크

데이터 전처리 파이프라인 (핵심)

HWP → LLM 정제 → 수작업 검수 (Human-in-the-loop)

AI의 성능은 데이터가 결정합니다. 학교의 HWP 문서 내 '표'나 '이미지'가 유실되는 문제를 방지하기 위해, LLM의 1차 구조화 후 행정실 담당자가 최종 검수하는 안전망(Human-in-the-loop) 파이프라인을 필수적으로 운영합니다.

📄 HWP 텍스트 추출 🤖 LLM Markdown 정제 👀 담당자 수동 검수 🗄️ 벡터 DB 형상 관리
기술 hwp5txt + Qwen 2.5
