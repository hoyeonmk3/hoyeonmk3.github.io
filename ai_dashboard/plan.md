# 국제대학 학과 소개 AI 포탈 — 구축 계획 초안

> 후보 3 기반. 8개 학과 정보를 AI가 자동 생성·번역하여 전시하는 소개 페이지.
> Google AI Ultra Pro 활용.

---

## 1. 프로젝트 목표

국제대학 8개 학과를 **한눈에** 소개하는 다국어 웹 포탈. AI가 학과 정보를 기반으로 소개 텍스트를 자동 생성하고 3개 언어로 번역.

### 핵심 차별점
- 기존 학교 홈페이지: 정적 텍스트, 한국어 중심
- AI 포탈: **Gemini가 학과 정보를 자연스러운 소개글로 재구성** + 한/영/중 자동 제공

### 프로젝트 1(RAG 챗봇)과의 관계
| | 챗봇 (프로젝트 1) | 포탈 (이 프로젝트) |
|---|---|---|
| 형태 | 질문-답변 (대화형) | 한눈에 보기 (전시형) |
| 사용 방식 | 구체적 궁금한 것 질문 | 학과 전체 훑어보기 |
| 상호 연결 | 포탈에서 "더 궁금하면 AI에게 물어보세요" → 챗봇 연결 |

## 2. 콘텐츠 구성

| 항목 | 내용 | 형태 |
|---|---|---|
| **학과 소개** | 학과 설명, 특색, 교육 목표 | AI 생성 텍스트 (한/영/중) |
| **주요 과목 소개** | 교과목 목록 + 과목별 설명 | 카드형 |
| **재학생 수 / 졸업생 수** | 단순 숫자 | 아이콘 + 숫자 |
| **현장실습 소개** | 실습 프로그램, 협력 기관, 사례 | 텍스트 + 사진 |
| **학과 행사** | 행사 목록, 소개 | 카드 or 목록 |

## 4. AI의 역할

| 용도 | 입력 | 출력 |
|---|---|---|
| **학과 소개글 생성** | 학과 기본 정보 (교육과정, 목표 등) | 자연스러운 소개 문단 |
| **과목 설명 생성** | 과목명 + 간단 키워드 | 과목별 1-2줄 설명 |
| **다국어 번역** | 한국어 원문 | 영어, 중국어 버전 |
| **행사 요약** | 행사명 + 일시 + 메모 | 한 줄 소개 |

### Gemini 프롬프트 예시
```
다음 학과 정보를 바탕으로, 유학생 대상 학과 소개글을 
자연스럽고 친근한 톤으로 작성해주세요. 3-4문장.

학과명: 글로벌경영학과
교육 목표: 국제 비즈니스 전문가 양성
주요 분야: 마케팅, 재무, 국제무역
특색: 해외 인턴십, 다국적 기업 연계

[한국어로 작성 후 영어, 중국어 버전도 함께 생성]
```

## 5. 기술 구성

| 구성 | 도구 | 역할 |
|---|---|---|
| **콘텐츠 생성** | Gemini API | 소개글 + 과목 설명 + 번역 |
| **데이터 관리** | Google Sheets | 학과 정보 원본 (행정 담당자 입력) |
| **이미지** | 학과 제공 사진 | 현장실습, 행사, 캠퍼스 |
| **웹** | 정적 사이트 (HTML/CSS/JS) | 홈페이지 임베딩 가능 |
| **호스팅** | GitHub Pages or 학교 서버 | 무료 or 기존 인프라 |

### 콘텐츠 생성 흐름
```
Google Sheets (학과 정보 원본)
       ↓ 스크립트 실행
Gemini API → 소개글/과목설명/번역 자동 생성
       ↓
정적 HTML 파일로 빌드
       ↓
호스팅 (홈페이지 임베딩)
```

> **핵심**: 실시간 API 호출 아님. 콘텐츠를 **사전 생성**하여 정적 페이지로 배포.
> → 운영 중 토큰 비용 0원. 데이터 변경 시에만 재생성.

## 7. 리스크

| 리스크 | 영향 | 대응 |
|---|---|---|
| 학과 정보/사진 미제공 | 콘텐츠 빈약 | 홈페이지 기존 정보 + 샘플 사진으로 MVP |
| AI 소개글 톤 불일치 | 학과마다 느낌 다름 | 통일된 프롬프트 템플릿 사용 |
