# 학과 홍보 콘텐츠 AI 생성기 — 구축 계획 초안

> 후보 4 기반. Google AI Ultra Pro 단독 활용.
> 학과 사진 + 캐릭터 아바타를 활용한 다국어 홍보 콘텐츠 자동 생성.

---

## 1. 프로젝트 목표

학과별 홍보 콘텐츠(이미지·텍스트·음성·영상)를 AI로 자동 생성하여, 외부에 "AI 활용 대학" 이미지를 보여주는 가시적 산출물 제작.

### 핵심 산출물
- 학과별 **캐릭터 아바타**가 학과를 소개하는 다국어 콘텐츠
- 형태: 영상 / 인터랙티브 웹 / SNS 게시물

### 운영 요건
| 요건 | 내용 |
|---|---|
| 도구 | **Google AI Ultra Pro 단독** (외부 유료 서비스 없음) |
| 언어 | 한국어, 영어, 중국어 |
| 입력 | 학과 사진 + 학과 정보 텍스트 + 아바타 예시 |
| 출력 | 학과별 홍보 콘텐츠 (영상/이미지/텍스트) |

---

## 2. Google AI Ultra Pro 활용 가능 기술

| 기술 | 용도 | Ultra Pro 포함 |
|---|---|---|
| **Gemini 2.5 Pro** | 홍보 텍스트 생성, 다국어 번역, 스크립트 작성 | ✅ |
| **Imagen 3** | 캐릭터 아바타 생성, 학과 이미지 스타일 변환 | ✅ |

| **Google Cloud TTS** | 다국어 음성 합성 (한/영/중) | ✅ (API) |
| **NotebookLM** | 학과 자료 정리/요약 | ✅ |
| **Google AI Studio** | 프롬프트 테스트 + API 키 관리 | ✅ |

## 3. 아바타 구현 방식 비교

### 옵션 A: AI 생성 정적 아바타 + TTS 음성

```
Imagen 3로 캐릭터 생성 → 정적 이미지 + TTS 음성 오버레이 → 슬라이드 영상
```

| 항목 | 평가 |
|---|---|
| 구현 난이도 | ★☆☆☆☆ 매우 쉬움 |
| 시각적 임팩트 | ★★★☆☆ 정적이라 약함 |
| Google 단독 | ✅ 100% |
| 제작 시간 | 학과당 1-2시간 |

---

### 옵션 B: AI 아바타 + 립싱크 영상 (Veo 2 활용)

```
Imagen 3으로 캐릭터 → Gemini로 스크립트 → TTS 음성 → Veo 2로 립싱크 영상
```

| 항목 | 평가 |
|---|---|
| 구현 난이도 | ★★★☆☆ 중간 |
| 시각적 임팩트 | ★★★★★ "AI가 말하는 영상" |
| Google 단독 | ✅ (Veo 2 립싱크 지원 여부 확인 필요) |
| 제작 시간 | 학과당 반나절 |

> ⚠️ **Veo 2의 캐릭터 립싱크 품질은 사전 테스트 필수**. 
> 일관된 캐릭터 유지가 어려울 수 있음.

---

### 옵션 C: 버츄얼 아바타 (VTuber 방식)

```
캐릭터 디자인 (Imagen 참고) → Live2D 모델링 → TTS 연동 → 실시간/녹화
```

| 항목 | 평가 |
|---|---|
| 구현 난이도 | ★★★★☆ 높음 (Live2D 필요) |
| 시각적 임팩트 | ★★★★★ 자연스러운 움직임 |
| Google 단독 | ❌ Live2D는 별도 도구 (VTube Studio 등) |
| 제작 시간 | 캐릭터 1개당 1-2주 (모델링) |

**하이브리드 접근 가능**:
- 캐릭터 **디자인**은 Imagen 3로 생성
- **음성**은 Google Cloud TTS
- **애니메이션**만 Live2D/VTube Studio (무료 도구 활용)
- → Google AI가 콘텐츠 생성, 외부 도구는 애니메이션 렌더링만

## 4. 제안 워크플로우

### Step 1: 캐릭터 아바타 콘셉트 확정
- 알버트가 아바타 예시 참고자료 제공
- Imagen 3로 학과별 캐릭터 프로토타입 생성 (3-5종)
- 스타일 확정: 일러스트 / 3D풍 / 애니메이션풍

### Step 2: 학과 자료 수집
- 학과 사진 수집 (캠퍼스, 실습실, 수업 장면 등)
- 학과 소개 텍스트
- Gemini로 다국어 홍보 스크립트 자동 생성

### Step 3: 콘텐츠 파이프라인 구축
```
[학과 사진 + 정보]
       ↓
[Gemini] → 홍보 스크립트 (한/영/중) 자동 생성
       ↓
[Imagen 3] → 학과 캐릭터 아바타 + 배경 이미지
       ↓
[Google TTS] → 다국어 음성 합성
       ↓
[Veo 2 or 편집] → 최종 영상/콘텐츠 조합
       ↓
[산출물]: 학과별 홍보 영상 / 이미지 / 웹 페이지
```

### Step 4: 아바타 방식 결정 (테스트 후)
- 옵션 B(Veo 2) 품질 테스트
- 품질 충분 → B 채택 (Google 단독)
- 품질 부족 → C(VTuber) 하이브리드 전환

### Step 5: 8개 학과 확장 생산
- 파이프라인 확정 후 학과별 반복 생산
- 학과당 캐릭터 + 스크립트 + 영상 세트

---

## 5. 일정 (예상)

```
Week 1  ─── 아바타 콘셉트 + Imagen 프로토타입 + Veo 2 테스트
Week 2  ─── 학과 자료 수집 + 스크립트 생성 + 파이프라인 확정
Week 3  ─── 2-3개 학과 시범 콘텐츠 제작
Week 4  ─── 나머지 학과 확장 + 최종 편집 + 게재
```
